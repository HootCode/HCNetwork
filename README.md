## Installation

HCNetworking is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following lines to your Podfile:

    
    # Add external source
    source 'https://github.com/HootCode/HCNetwork.git'

    # Add repo
    pod "HCNetwork"
